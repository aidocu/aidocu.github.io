设置和预设

/settings 命令提供了常用选项的切换按钮，例如模型版本、样式值、质量值和放大版本。 设置还具有 /stealth 和 /public 命令的切换。

添加到提示末尾的参数将覆盖使用 /settings 所做的选择。

模型版本下拉列表​

选择生成图像时要使用的Midjourney 模型版本。 使用最新模型是默认选择，并且将始终使用最新的 Midjourney 模型版本。

Use the default Model (V6) Midjourney Model V6[ALPHA] 🌈 Niji Model V6[ALPHA] 5️⃣ Mjourney Model V5.2 5️⃣ Mjourney Model V5.1 🍎 Niji Model V5 5️⃣ Mjourney Model V5.0 🌈 Niji Model V4 4️⃣ Mjourney Model V4 3️⃣ Mjourney Model V3 2️⃣ Mjourney Model V2 1️⃣ Mjourney Model V1

样式原始参数​

Midjourney 模型版本 5.1、5.2、6 和 Niji 6 可以使用 --style raw 参数进行微调，以减少 Midjourney 默认审美。 如果选择其他模型版本，则此切换不可用。

🔧 原始模式

风格化参数​

Midjourney 机器人经过训练，可以生成偏爱艺术色彩、构图和形式的图像。 --stylize 或 --s 参数影响此次训练的应用程度。 低风格化值生成的图像与提示非常匹配，但艺术性较差。 高风格化值创建的图像非常艺术，但与提示的联系较少。

🖌️ Stylize Low 🖌️ Stylize Med 🖌️ Stylize High 🖌️ Stylize Very High

Stylize Low = --s 50, Stylize Med = --s 100, Stylize High = --s 250, Stylize Very High = --s 750,

公开和隐身模式​

在公开模式和隐身模式之间切换。 对应于 /public 和 /stealth 命令。

🧍‍♂️Public

再混合模式​

使用再混合模式更改提示、参数、模型版本或变体之间的长宽比。 Remix 将采用起始图像的总体构图并将其用作新作业的一部分。 重新混合可以帮助改变图像的设置或照明、发展主题或实现棘手的构图。

🎛️ Remix

高低变化模式​

在高变化和低变化模式之间切换。

🎨 High Variation Mode 🎨 Low Variation Mode

粘性风格​

粘性样式将保存您个人后缀中使用的最后一个 --style 代码参数，因此您不必在将来的提示中重复该代码。 通过使用新的 --style 或取消选择粘性样式来更改代码。

📌 Sticky Style

加速、快速和放松模式​

在 Turbo、Fast 和 Relaxed 模式之间切换。 对应于 /turbo、/fast 和 /relax 命令以及 --turbo、--fast 和 --relax 参数。

⚡ Turbo Mode 🐇 Fast Mode 🐢 Relax Mode

重置设置​

返回默认设置。

Reset Settings

自定义首选项​

使用prefer 命令创建自定义选项，以将常用参数自动添加到提示末尾。
/prefer option 创建或管理自定义选项。
/prefer option list 查看当前的自定义选项。
/prefer suffix 指定添加到每个提示末尾的后缀。

自动私信结果 使用 /prefer auto_dm 打开或关闭将完成的作业自动发送到您的私信。

首选选项​

/prefer option set <name> <value> 命令允许您创建自定义快捷方式，向提示添加多个参数或文本片段。

/prefer option set mine --chaos 5 --aspect 2:3 创建一个名为“mine”的选项，其中包含混乱和长宽比参数。


使用 /imagine prompt vibrant California poppies --mine，被解释为 /imagine prompt vibrant California poppies --chaos 5 --aspect 2:3.

/prefer option list 列出使用 prefer 选项集 创建的所有选项。 用户最多可以有 20 个自定义选项。

要删除自定义选项，请使用 /prefer option set <要删除的名称> 并将值字段留空。

首选后缀​

/prefer suffix 在所有提示后自动附加指定的后缀。
使用 /settings 命令并选择 Reset Settings 以清除首选后缀。

命令示例：/prefer suffix --uplight --video
