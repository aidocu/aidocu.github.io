用户信息

使用 /info 命令查看有关当前排队和正在运行的作业、订阅类型、续订日期等的信息。

User ID​

您的Midjourney 用户 ID

Subscription​

订阅部分显示您订阅的计划以及您的下一次续订日期。

Visibility Mode​

显示您当前处于公共模式还是隐身模式。 隐身模式仅适用于 Pro Plan 订阅者。

Fast Time Remaining​

显示当月剩余的 Fast GPU 时间。 快速 GPU 时间每月重置一次并且不会结转。

Ranking Count​

显示您排名的图像总数，以及过去 30 天内排名的图像数量。

Lifetime Usage​

显示您一生的Midjourney 统计数据。 图像包括所有类型的生成（初始图像网格、放大、变化、再混合等）。

Fast, Turbo and Relaxed Usage​

显示您的快速、加速和轻松模式使用情况。

Queued Fast and Relaxed Jobs​

列出所有排队运行的作业。

Running Jobs​

列出当前正在运行的所有作业。 最多可以同时运行三个作业。

使用信息命令​

在任何机器人频道或您的私信中输入 /info。 只有您才能看到您的信息弹出窗口。
