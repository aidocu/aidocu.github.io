命令列表

您可以通过键入命令与 Discord 上的 Midjourney Bot 进行交互。 命令用于创建图像、更改默认设置、监视用户信息以及执行其他有用的任务。

Midjourney 命令可以在任何 机器人频道、允许Midjourney 机器人运行的私有 Discord 服务器上或在与Midjourney 机器人的私信中使用。

命令​

/ask​

获得问题的答案。

/blend​

轻松将两个图像混合在一起。

/daily_theme​

切换 #daily-theme 频道更新的通知 ping

/Describe​

根据您上传的图像编写四个示例提示。

/docs​

在官方 Midjourney Discord 服务器中使用可快速生成本用户指南中涵盖的主题的链接！

/fast​

切换到快速模式。

/feedback​

提交有关 Midjourney 的建议和想法并对其他人的想法进行评分。

/faq​

在官方 Midjourney Discord 服务器中使用可快速生成流行提示工艺频道常见问题解答的链接。

/help​

显示有关Midjourney 机器人的有用基本信息和提示。

/imagine​

使用提示生成图像

/info​

查看有关您的帐户以及任何排队或正在运行的作业的信息。

/invite​

获取 Midjourney Discord 服务器的邀请链接。

/list_tuners​

生成您创建的所有风格调音器的列表。

/prefer option set​

创建或管理自定义选项。

/prefer option list​

查看您当前的自定义选项。

/prefer remix​

切换再混合模式。

/prefer suffix​

指定要添加到每个提示末尾的后缀。

/prefer variability​

使用图像网格下方的 V1 V2 V3 和 V4 按钮时，在高和低变化之间切换。

/public​

对于专业计划订阅者：切换到公共模式

/relax​

切换到放松模式。

/settings​

查看和调整Midjourney 机器人的设置

/stealth​

对于专业计划订阅者：切换到隐身模式

/shorten​

提交长提示并接收有关如何使其更简洁的建议。

/show​

使用图像作业 ID 在 Discord 中重新生成作业。

/subscribe​

生成用户帐户页面的个人链接。

/synonyms​

在官方 Midjourney Discord 服务器中使用来探索相关单词和短语以在提示中尝试。

/tune​

根据您的提示生成风格调谐器。 风格调整器可让您打造自己的Midjourney 风格并自定义工作的外观。

/turbo​

切换到Turbo模式。

/userid​

获取您的Midjourney 用户 ID
