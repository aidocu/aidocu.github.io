Blend混合

/blend 命令允许您快速上传 2-5 张图像，然后查看每个图像的概念和美学，并将它们合并成一个新颖的新图像。

  • /blend 与通过 /imagine 使用多个图像提示相同，但界面经过优化，以便在移动设备上轻松使用。
  • /blend 最多可处理 5 个图像。 要在提示中使用超过 5 个图像，请使用带 /imagine 的图像提示
  • /blend 不适用于文本提示。 要同时使用文本和图像提示，请将图像提示和文本通过 /imagine 一起使用

/blend 选项

输入 /blend 命令后，系统将提示您上传两张照片。 使用移动设备时，从硬盘驱动器拖放图像或从照片库添加图像。 要添加更多图像，请选择 optional/options 字段并选择 image3、image4 或 image5。 /blend 命令可能比其他命令需要更长的时间来启动，因为您的图像必须先上传，然后 Midjourney Bot 才能处理您的请求。

混合图像的默认长宽比为 1:1，但您可以使用 dimensions 字段在方形长宽比 (1:1)、纵向长宽比 (2:3) 或横向长宽比 (3:2) 之间进行选择 。

与任何其他 /imagine 提示一样，自定义后缀会添加到 /blend 提示的末尾。 作为 /blend 命令一部分指定的长宽比会覆盖自定义后缀中的长宽比。

混合技巧
为获得最佳效果，请上传与所需结果具有相同宽高比的图像。
