平移

“平移”选项允许您沿选定方向扩展图像的画布，而不更改原始图像的内容。 新扩展的画布将使用提示和原始图像的指导进行填充。

info

平移允许您将图像分辨率提高到超出单个方向最大 1024 像素 x 1024 像素尺寸。
Pan 与 Midjourney 模型版本 5、5.1、5.2 和 niji 5 兼容

平移图像​

平移按钮 ⬅️ ➡️ ⬆️ ⬇️ 将在放大图像后出现。 平移时，仅使用距离图像侧面最近的 512 个像素以及提示来确定新部分。

Prompt: A vibrant fantasy landscape

平移图像一次后，您只能沿同一方向（水平/垂直）再次平移该图像。 您可以根据需要继续朝该方向平移。 平移多次后，图像可能会变得太大而无法在 Discord 上发送。 发生这种情况时，您将收到一个图像链接。

平移 ⬅️ 和 ➡️ 多次

再混合模式下的平移

平移支持再混合模式。 这允许您在平移时更改提示。
