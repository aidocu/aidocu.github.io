参数列表

参数是添加到提示中的选项，用于更改图像的生成方式。 参数可以更改图像的长宽比、在Midjourney 模型版本之间切换、更改使用的 Upscaler 等等。

参数始终添加到提示的末尾。 您可以向每个提示添加多个参数。

使用苹果设备？
许多 Apple 设备会自动将双连字符 (-) 更改为长破折号 (—)。 Midjourney 接受两者！

基本参数​

Aspect Ratios​

--aspect 或 --ar 更改生成的长宽比。

Chaos​

--chaos <number 0–100> 更改结果的变化程度。 更高的价值观会产生更多不寻常和意想不到的世代。

Fast​

--fast 覆盖您当前的设置并使用快速模式运行单个作业。

Image Weight​

--iw <0–2> 设置相对于文本粗细的图像提示粗细。 默认值为 1。

No​

--no 负面提示，--no plants 会尝试从图像中删除植物。

Quality​

--quality <.25、.5 或 1>，或 --q <.25、.5 或 1> 您想要花费多少渲染质量时间。 默认值为 1。值越高，使用的 GPU 分钟数越多； 较低的值使用较少。

Random​

--style random，将随机的 32 种基本样式 Style Tuner 代码添加到您的提示中。 您还可以使用 --style random-16、--style random-64 或 --style random-128 来使用来自其他长度的风格调音器的随机结果。

Relax​

--relax 覆盖您当前的设置并使用放松模式运行单个作业。

Repeat​

--repeat <1–40> 或 --r <1–40> 从单个提示创建多个作业。 --repeat 对于快速重新运行作业多次很有用。

Seed​

--seed <0–4294967295 之间的整数> Midjourney 机器人使用种子号来创建视觉噪声场（如电视静态），作为生成初始图像网格的起点。 种子数是为每个图像随机生成的，但可以使用 --seed 或 --sameseed 参数指定。 使用相同的种子编号和提示将产生相似的结局图像。

Stop​

--stop <10–100 之间的整数> 使用 --stop 参数可在进程Midjourney 完成作业。 以较早的百分比停止作业可能会产生更模糊、不太详细的结果。

Style​

--style <raw> 在Midjourney 模型版本 5.1 和 5.2 的版本之间切换。
--style <4a、4b 或 4c> 在 Midjourney 模型版本 4 的版本之间切换。
--style <可爱、富有表现力、原创或风景> 在 Niji 模型版本 5 的版本之间切换。
使用 /tune 命令创建样式调谐器并生成自定义样式代码。

Stylize​

--stylize <number> 或 --s <number> 参数影响 Midjourney 的默认审美风格应用于作业的强度。

Tile​

--tile 参数生成可用作重复图块以创建无缝图案的图像。

Turbo​

--turbo 覆盖您当前的设置并使用 Turbo 模式运行单个作业。

Video​

--video 保存正在生成的初始图像网格的进度视频。 表情符号会对完成的图像网格做出反应，并使用 ✉️ 触发将视频发送到您的直接消息。 --放大图像时视频不起作用。

Weird​

--weird <数字 0–3000> 或 --w <数字 0–3000> 使用实验性的 --weird 参数探索不寻常的美学。

模型版本参数​

Midjourney 定期发布新模型版本，以提高效率、一致性和质量。 不同的模型擅长处理不同类型的图像。

Niji​

--niji <4, 或 5> 专注于动漫风格图像的替代模型。

Version​

--version <1、2、3、4、5.0、5.1、5.2 或 6> 或 --v <1、2、3、4、5.0、5.1、5.2 或 6> 使用不同版本的 Midjourney 算法。

旧参数​

某些参数仅适用于早期的Midjourney 模型。 在此处阅读有关旧参数的更多信息。
