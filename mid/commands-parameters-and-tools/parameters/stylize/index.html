风格化

Midjourney 机器人经过训练，可以生成有利于艺术色彩、构图和形式的图像。 --stylize 或 --s 参数影响此训练的应用程度。 低风格化值生成的图像与提示非常匹配，但艺术性较差。 高风格化值创建的图像非常艺术，但与提示的联系较少。

--stylize 的默认值为 100，使用当前模型时接受 0–1000 的整数值

常用风格化设置​

Stylize 对模型版本 6 的影响​

提示示例：/imagine prompt child's drawing of a cat --s 100

提示示例：/imagine prompt colorful risograph of a fig --s 100

如何切换风格化值​

使用风格化参数​

将 --stylize <value> 或 --s <value> 添加到提示末尾。

使用设置命令​

输入 /settings 并从菜单中选择您喜欢的风格值。

🖌️ Style Low 🖌️ Style Med 🖌️ Style High 🖌️ Style Very High
