探索提示

即使是简短的单字提示也会在Midjourney的默认风格中产生美丽的图像，但您可以通过结合艺术媒介、历史时期、地点等概念来创建更有趣的个性化结果。

选择一种媒介​

拆开油漆、蜡笔、刮板、印刷机、闪光粉、墨水和彩纸。 生成时尚图像的最佳方法之一是指定艺术媒介。

提示示例：/imagine prompt <any art style> style cat

获取具体信息​

更精确的单词和短语将有助于创建具有完全正确的外观和感觉的图像。

提示示例：/imagine prompt <style> sketch of a cat

时间旅行​

不同的时代有不同的视觉风格。

提示示例：/imagine prompt an illustration of a <time period> cat

表情​

使用情感词语赋予人物个性。

提示示例：/imagine prompt <emotion> cat

变得多彩​

全方位的可能性。

提示示例：/imagine prompt <color word> colored cat

环境探索​

不同的环境可以设定独特的情绪。

提示示例：/imagine prompt <location> cat
