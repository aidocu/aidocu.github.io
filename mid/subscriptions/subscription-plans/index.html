订阅计划

Midjourney 有四个订阅级别。 按月或全年支付可享受 20% 的折扣。 每个订阅计划都包括访问 Midjourney 会员图库、官方 Discord、一般商业使用条款等。

note

如何订阅 使用 /subscribe 命令生成订阅页面的个人链接。
或者，访问 Midjourney.com/account。
或者，在登录 Midjourney 网站时从侧边栏选择 Manage Sub。

订阅计划​

访问 https://www.midjourney.com/account/ 或使用 /subscribe 命令生成订阅页面的链接。

不要分享您的链接
生成的链接对于每个帐户都是唯一的，切勿与其他人共享！

支付方式​

目前仅接受 Stripe 支持的付款方式：Mastercard、VISA 或 American Express 等服务机构发行的信用卡或借记卡。 Stripe 是 PCI 1 级服务提供商，这是支付行业中最严格的认证级别。
Google Pay、Apple Pay 和 Cash App Pay 在某些地区可用。
不支持 PayPal、电汇和类似方法。

管理你的计划​

在 https://www.midjourney.com/account/ 管理您的订阅计划。

每月续订​

  • 每月未使用的 Fast GPU 时间会被清空。
  • 续订时，快速模式会自动重新激活。
  • 您在我们的第三方支付处理机构备案的付款方式将在每个月度或年度计费周期结束时自动收取费用。

切换计划​

随时升级您的计划。 升级时，您可以选择升级是立即生效还是在当前计费周期结束时生效。 如果您选择立即升级，我们将根据您要升级的计划的使用情况向您提供按比例分配的价格。 降级始终在当前计费周期结束时生效。

note

按比例分配？
当您选择立即升级时，未使用的快速 GPU 时间将计入您的帐户。 如果 Basic 订阅者在不使用任何 GPU 时间的情况下立即升级，他们将获得 10 美元的新套餐积分。 如果标准订阅者在使用了 50% 的快速 GPU 时间后进行升级，他们将获得 15 美元的新计划积分。


取消​

请前往 https://www.midjourney.com/account/ 随时取消订阅。 取消在当前计费周期结束时生效。 订阅权益（例如访问社区图库和批量下载工具）在当前计费周期结束之前均可用。

如果您取消订阅，您生成的图像和 Midjourney.com/account 页面不会被删除。 您可以随时访问 https://www.midjourney.com/account/ 重新订阅计划。

退款​

对于终身使用 GPU 分钟（包括放松模式下使用的时间）少于 20 分钟的订阅者，可以退款。 如果您符合退款条件，当您点击取消帐户时，会自动弹出该对话框。

购买更多Fast​

快速 GPU 耗尽并想在每月订阅续订之前购买更多？ 您可以以 4 美元/小时的价格购买额外的 Fast GPU 小时数。 每小时价格是实验性的，可能会发生变化。

在您的 Midjourney 帐户页面上购买更多 Fast GPU 小时数。

购买的 Fast GPU 小时数不会过期，但您必须拥有有效的订阅才能使用它们。

快速时间不予退款。

帐户信息​

使用 /info 命令查看有关您帐户的信息，包括订阅期剩余的 快速时间、生命周期使用统计信息、排队或正在运行的作业数量以及订阅续订日期。
