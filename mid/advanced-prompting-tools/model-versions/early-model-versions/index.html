早期模型版本

Midjourney 定期推出新模型版本，以提高一致性、效率、质量和风格。 您可以使用 /settings 命令并选择您喜欢的模型版本来切换模型版本。 不同的模型擅长生成不同类型的图像。

在此处了解最新的Midjourney 模型

模型版本 1、2 和 3（旧版）​

您可以使用 --version 或 --v 参数或使用 /settings 命令并选择模型版本来访问早期的中间模型。 不同的模型擅长处理不同类型的图像。

提示示例：/imagine prompt vibrant California poppies --v 1

--version 3
default model: 07/22–11/22
极具创意的作品
中等连贯性

--version 2 default model: 04/22–07/22
富有创意、色彩丰富、具有绘画性
低一致性

--version 1
default model: 02/22–04/22
非常抽象和绘画性
低一致性

--hd (high definition)
早期替代模型
繁忙的细节和抽象
低一致性

Niji Model 4（旧版）​

niji 4 模型是 Midjourney 和 Spellbrush 之间的合作，经过调整可以制作动画和插画风格。 --niji 4 模型对动漫、动漫风格和动漫美学有更多的了解。 一般来说，它非常适合动态和动作镜头以及以角色为中心的构图。

提示示例：/imagine prompt vibrant California poppies --niji 4

--v 4
California poppies --v 4

--niji 4
California poppies --niji 4

--v 4
birds sitting on a twig --v 4

--niji 4
birds sitting on a twig --niji 4

测试模型（旧版）​

有时会临时发布新模型以供社区测试和反馈。 目前有两种可用的测试模型：--test 和 --testp，可以与**--creative**参数结合使用，以获得更多样的作品。

提示示例：/imagine prompt vibrant California poppies --testp --creative

--test
具有良好连贯性的通用艺术模型

--test + --creative

--testp
具有良好一致性的照片写实模型

--testp + --creative

note

当前测试模型的注释 --test 和 --testp
测试模型仅支持 1250–5000 之间的 --stylize 值。
测试模型不支持多重提示或图像提示
测试模型的最大纵横比为 3:2 或 2:3。
当长宽比为 1:1 时，测试模型仅生成两个初始网格图像。
当长宽比不是 1:1 时，测试模型仅生成一张初始网格图像。
提示前面的单词可能比后面的单词更重要。

早期模型放大工具​

早期的 Midjourney 模型版本首先为每个作业生成低分辨率图像选项网格。 您可以在任何这些图像上使用 Midjourney upscaler 来增加尺寸并添加其他细节。 有多种高档模型可用于放大图像。 使用放大程序会占用您订阅的 GPU 分钟数。

每个图像网格下方的 U1 U2 U3 U4 按钮用于放大所选图像。

了解有关当前放大选项的更多信息

所有尺寸均适用于长宽比为 1:1 的正方形。

轻型放大器​

Light Upscaler 创建 1024 像素 x 1024 像素图像并添加适量的细节和纹理。 使用较旧的 Midjourney 模型版本时，Light Upscaler 对于面和光滑表面非常有用。

使用 --uplight 参数更改 U1 U2 U3 U4 upscale 按钮的行为以使用 Light Upscaler。

prompt: adorable rubber duck medieval knight

prompt: sand cathedral

详细放大器​

Detail Upscaler 创建 1024px x 1024px 图像并添加许多图像的额外细节。

可以使用 Upscale to Max (放大到最大)按钮再次放大已使用详细放大器放大的图像，以获得 1664 像素 x 1664 像素的最终分辨率。 放大到最大仅在快速模式下可用。

详细放大器是 Midjourney 模型版本 V1、V2、V3 和 hd 的默认设置

prompt: adorable rubber duck medieval knight

prompt: sand cathedral

Beta 放大器​

Beta Upscaler 可创建 2048 像素 x 2048 像素的图像，而无需添加许多其他细节。 Beta Uspcaler 对于面部和光滑表面非常有用。

使用 --upbeta 参数更改 U1 U2 U3 U4 放大按钮的行为以使用 Beta Upscaler。

prompt: adorable rubber duck medieval knight

prompt: sand cathedral
