旧模型版本参数

参数是添加到提示中的选项，用于更改图像的生成方式。 参数可以更改图像的长宽比、在Midjourney 模型版本之间切换、更改使用的放大器 等等。

此页面上的参数仅适用于旧的 Midjourney 模型版本。 有关适用于最新 Midjourney 模型版本的参数的信息，请在此处阅读更多信息。

旧模型版本参数​

Midjourney 定期发布新模型版本，以提高效率、一致性和质量。 不同的模型擅长处理不同类型的图像。

Niji​

--niji 4 和 niji 5 使用专注于动漫风格图像的替代模型。

High Definition​

--hd 使用早期的替代模型，生成更大、不太一致的图像。 该算法可能适用于抽象和风景图像。

Test​

--test 使用 Midjourney 特殊测试模型。

Testp​

--testp 使用 Midjourney 特殊摄影测试模型。

Version​

--version <1、2、3、4 或 5> 或 --v <1、2、3、4 或 5> 使用不同版本的 Midjourney 算法。 当前算法是默认设置。

传统放大器参数​

旅程中首先为每个作业生成一个低分辨率图像选项网格。 您可以在任何网格图像上使用 Midjourney 放大器来增加尺寸并添加其他细节。 有多种高档模型可用于放大图像。

Uplight​

--uplight 选择 U 按钮时使用替代的“浅”放大器。 结果更接近原始网格图像。 放大后的图像细节较少且更平滑。

Upbeta​

--upbeta 选择 U 按钮时使用替代 beta 放大器。 结果更接近原始网格图像。 放大后的图像添加的细节明显减少。

旧参数​

这些参数仅适用于特定的早期Midjourney 模型。

Creative​

--creative 修改 test 和 testp 模型，使其更加多样化和创造性。

Sameseed​

--sameseed 种子值创建一个应用于初始网格中所有图像的单个大型随机噪声场。 当指定 --sameseed 时，初始网格中的所有图像都使用相同的起始噪声，并将生成非常相似的生成图像。
