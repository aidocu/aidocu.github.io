怪异

使用实验性的 --weird 或 --w 参数探索非传统美学。 此参数为生成的图像引入了古怪和另类的品质，从而产生独特和意想不到的结果。

info

--weird 接受值：0–3000。
默认 --weird 值为 0。
--weird 是一个高度实验性的功能。 奇怪的事情可能会随着时间的推移而改变 --weird 与 Midjourney 模型版本 5、5.1、5.2、6、niji 5 和 niji 6 兼容
--weird 与种子不完全兼容

怪异对工作的影响​

最佳的 --weird 值取决于提示并需要进行实验。 尝试从较小的值开始，例如 250 或 500，然后从那里向上/向下。 如果您希望一代人具有传统的吸引力和怪异，请尝试将更高的 --stylize 值与 --weird 混合。 尝试从两者相似的值开始。 例如: /imagine prompt cyanotype cat --stylize 250 --weird 250

提示示例：/imagine prompt cyanotype cat --weird 250

提示示例：/imagine prompt lithograph potato --weird 250

提示示例：/imagine prompt clockwork chicken --weird 250

--weird、--chaos 和 --stylize 之间有什么区别？​

--chaos 控制初始网格图像彼此之间的差异程度。 --stylize 控制Midjourney 默认美学的应用程度。 --weird 控制图像与之前的Midjourney 图像相比的异常程度。

如何使用怪异参数​

将 --weird <value> 或 --w <value> 添加到提示符末尾。
